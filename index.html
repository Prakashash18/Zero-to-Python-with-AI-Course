Step 1 · Learn Smart

What is AI — and why it helps

An AI helper (Claude, ChatGPT, Gemini, Copilot) does two things for a beginner: it turns plain English into Python code, and it explains code back to you in plain English. You don't memorise syntax — you learn to ask well and read the answer.

Think of it as a fast junior partner, not a magic oracle — it sometimes gets things wrong, which is exactly why we still learn to read code.

Step 1 · Learn Smart

How to ask: a good code prompt

A good prompt is the whole skill. Three habits get you most of the way:

  1. Be specific and complete. Say the goal, the inputs, and the output you want.
  2. Give context and purpose. Tell it you're a beginner and why, so it explains at the right level.
  3. Ask one thing at a time. Get it working, then ask for a single improvement.
Weak prompt
"make a counter"

The micro:bit is a tiny computer with a built-in accelerometer — a motion sensor, like in your phone. We'll use it to count steps. Same Python ideas from Step 1, now running in your hand.

Plug the micro:bit into your laptop with USB. Open MakeCode, click Connect, follow the pairing prompt, and flash the flashing-heart example first — once it shows on the LEDs, everything works.

Step 2 · Code Fast

2 · Build the step counter

Every line is a Step 1 idea in the wild. Flash it, strap the micro:bit to your ankle, and walk around.

from microbit import *

steps = 0              # the variable (a box)
while True:            # the loop (runs forever)
    if accelerometer.was_gesture('shake'):  # the if (a step!)
        steps += 1     # add one
        display.show(steps)

Does the count match your real steps? Almost never exactly — and that's what the next part fixes.

Step 3 · Build with Data

2 · Analyse it

Start with the data as a list, then squeeze out the big numbers:

days  = ["Mon","Tue","Wed","Thu","Fri","Sat","Sun"]
steps = [6200, 8100, 6400, 13420, 7300, 12500, 11200]

print("Total:", sum(steps))
print("Average:", round(sum(steps)/len(steps)))
print("Best day:", max(steps))

Then a text bar chart with the for loop — no chart library needed:

for i in range(len(days)):
    bar = "#" * (steps[i] // 1000)
    print(days[i], bar, steps[i])
